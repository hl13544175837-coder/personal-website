# 个人网站设计文档

**日期:** 2026-03-30  
**项目:** AI PM 个人作品集网站  
**风格:** 暗色极客风

---

## 1. 概念与愿景

一个面向面试官的暗色极客风个人网站，展示AI训练师背景、AI PM潜力和个人创作能力。整体给人"懂AI、爱折腾、有审美"的印象，适合申请AI产品经理岗位。

---

## 2. 设计语言

### 美学方向
暗色科技风，参考Vercel/Linear的极简美学 + 霓虹点缀

### 色彩系统
- 背景: `#0a0a0a`
- 卡片: `#111111`
- 边框: `#222222`
- 主色: `#00d4ff` (霓虹蓝)
- 辅助: `#00ff88` (霓虹绿)
- 文字: `#ffffff` / `#888888`

### 字体
- 标题: Geist / Outfit
- 正文: Geist
- 代码: JetBrains Mono

### 动效
- 卡片hover: 轻微上浮 + 边框发光
- 页面滚动: 淡入效果
- 克制、不花哨

---

## 3. 页面结构

### 3.1 Hero 区域
- 左侧: 个人照片（圆形裁剪）+ 一句话定位
- 右侧: 简短自我介绍（2-3行）
- 底部: 社交图标（GitHub / 抖音 / B站 / 邮箱）

### 3.2 关于我
- 标题: "About Me"
- 内容: 阿里AI训练师背景 → 转型AI PM → 学习路径
- 突出: AI工具使用经验、产品思维成长

### 3.3 经历
- 时间线形式展示
- 阿里实习（重点）
- AI工具学习历程
- 每条配简短说明

### 3.4 项目
- Bento网格布局
- CosCraft 卡片（AI生成Cosplay参考资料）
- 次元置换 卡片（二次元物物交换平台）
- 点击跳转详情（暂用锚点）

### 3.5 创作
- 标题: "Works"
- 抖音/B站视频嵌入展示
- 3-4个代表作品
- 支持多平台

### 3.6 简历
- 在线PDF预览
- 下载按钮
- 联系方式（邮箱 + 微信）

---

## 4. 技术方案

- **框架:** 纯 HTML + CSS + Vanilla JS
- **部署:** GitHub Pages（绑定火山引擎域名）
- **视频:** 抖音/B站 iframe 嵌入
- **动画:** CSS transitions + Intersection Observer
- **图标:** Lucide Icons（CDN）
